📋 СЛЕДУЮЩИЕ ШАГИ:

1️⃣  СКАЧАЙТЕ ДАТАСЕТ IMDB
   
   Перейдите на Kaggle:
   https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews
   
   Скачайте файл "IMDB Dataset.csv" (66 MB)
   Поместите его в папку: data/IMDB Dataset.csv

2️⃣  ОБУЧИТЕ МОДЕЛЬ (2-3 минуты)
   
   cd "/home/levi/Developer/SentiScan AI"
   ./train.sh
   
   Или вручную:
   source myenv/bin/activate
   python train.py

3️⃣  ЗАПУСТИТЕ СЕРВЕР
   
   ./run.sh
   
   Или вручную:
   source myenv/bin/activate
   python app.py

4️⃣  ОТКРОЙТЕ В БРАУЗЕРЕ
   
   http://localhost:5000
