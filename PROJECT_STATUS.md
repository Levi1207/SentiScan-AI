## 📋 Следующие шаги

### 1. Дождитесь завершения установки зависимостей
Установка библиотек запущена в фоне и скоро завершится.

### 2. Скачайте датасет IMDB
```bash
# Перейдите на Kaggle:
https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

# Скачайте файл "IMDB Dataset.csv" (66 MB)
# Поместите его в папку: data/IMDB Dataset.csv
```

### 3. Обучите модель
```bash
cd "/home/levi/Developer/SentiScan AI"
./train.sh
# или
source myenv/bin/activate
python train.py
```

Обучение займёт 2-3 минуты и создаст файлы:
- `models/vectorizer.pkl` — TF-IDF векторайзер
- `models/model.pkl` — Обученная Logistic Regression

### 4. Запустите сервер
```bash
./run.sh
# или
source myenv/bin/activate
python app.py
```

Сервер запустится на: http://localhost:5000

### 5. Откройте в браузере
Перейдите на http://localhost:5000 и начните анализировать тексты!

## 🎨 Возможности

### Веб-интерфейс
- 📝 Ввод текста для анализа
- 🎯 Определение тональности (позитивная/негативная)
- 📊 Уровень уверенности модели
- 🔑 Ключевые слова, повлиявшие на решение
- 📈 Вероятности для каждого класса
- 📜 История последних 10 анализов (сохраняется в localStorage)

### API Endpoints
- `POST /predict` — Анализ тональности текста
- `GET /health` — Проверка работоспособности
- `GET /stats` — Статистика модели

## 🔧 Технологии

- **Backend:** Python 3.13, Flask 3.1, scikit-learn 1.8
- **ML:** TF-IDF (10,000 features) + Logistic Regression
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Датасет:** IMDB Movie Reviews (50,000 отзывов)

## 📊 Ожидаемые метрики

После обучения модель должна показать:
- **Accuracy:** ~89-91%
- **Precision:** ~90%
- **Recall:** ~88%
- **F1-Score:** ~89%
- **AUC-ROC:** ~0.96
